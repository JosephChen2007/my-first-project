# Steam 今日特惠排行榜分析工具

這是一個使用 Python 開發的自動化爬蟲工具，旨在分析 Steam 平台上折扣幅度最大的遊戲。

## 🌟 核心功能
* **自動爬取**：即時獲取 Steam 特惠網頁數據。
* **資料清洗**：處理複雜的 HTML 標籤並將折扣轉為數字。
* **數據分析**：自動按折扣大小排序，找出 CP 值最高的遊戲。
* **視覺化**：繪製橫向長條圖，解決中文顯示亂碼問題。

## 🛠️ 使用技術
* Python 3.13
* Requests / BeautifulSoup4 (爬蟲)
* Matplotlib (資料視覺化)
* Re (正則表達式資料提取)
